# Sentiment Analysis with LSTM

A simple sentiment analysis model built with TensorFlow/Keras using LSTM layers, trained on a small dataset.

## Features

- Train a sentiment analysis model
- Web GUI using Streamlit for real-time predictions

## Installation

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the training script: `python sentiment_model.py`
4. Run the Streamlit app: `streamlit run streamlit_app.py`

## Usage

Enter text in the Streamlit app to get sentiment prediction (Positive/Negative) with confidence score.
